An action list is a simple list of interactive actions or options. It displays a single column of content, divided into items.

- **Action list item:** An action list item is a generic interactive element, which includes a main text and optional icons, descriptions, auxiliary information, and more.
- **Action list section:** An action list section uses the ARIA role property to group elements from a list. Depending on the provided values, it renders a divider, a header, or both.
